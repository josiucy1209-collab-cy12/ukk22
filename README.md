# Aplikasi Penjualan UKK 2022

Ini adalah aplikasi web sederhana untuk pengelolaan data barang dan transaksi penjualan yang dibuat dalam rangka memenuhi tugas **UKK (Uji Kompetensi Keahlian) 2022**.

## Fitur Utama
* Login Petugas (Tanpa Password)
* Manajemen Data Produk
* Transaksi Penjualan
* Hapus dan Edit Data

## Cara Instalasi
1. Clone repository ini.
2. Pindahkan ke folder `htdocs`.
3. Buat database bernama `ukk22` di phpMyAdmin.
4. Import file SQL (jika ada).
5. Sesuaikan konfigurasi di `koneksi.php`.
